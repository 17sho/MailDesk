# MailDesk v0.4.2

本版本简化邮件阅读链路：取件解析得到什么，阅读器就直接渲染什么，不再经过独立的网络图片下载子系统。

## 邮件直接显示

- 直接渲染提取到的 HTML 排版和图片地址。
- CID 内嵌图片、图片附件和 `data:` 图片继续使用邮件自身的数据。
- `http/https` 图片由 HTML 阅读器直接显示，不再要求用户点击加载按钮。
- 保留表格、行内样式、背景图片、延迟图片地址和常见邮件布局。

## 删除旧图片下载体系

- 删除 `RemoteImageWorker` 及其信号和线程任务。
- 删除独立网络图片下载、并发请求、文件嗅探和内存缓存模块。
- 删除“加载网络图片”按钮、状态条和自动加载图片设置。
- 删除相关样式、测试和已经没有调用方的图片地址收集死代码。

## 阅读器边界

- JavaScript、iframe、表单、插件、弹窗、下载和本地文件访问仍被禁用。
- 阅读器只允许 HTML 中的图片发起网络请求；外部脚本、样式表和其他非图片资源继续被拦截。
- 用户点击邮件链接时仍需要明确确认，并交给系统默认浏览器处理。

## 验证

- IMAP 仍使用完整 `RFC822` 邮件原件进行解析。
- Graph HTML、CID 附件与普通 MIME 邮件使用同一直接显示流程。
- 已覆盖主界面、独立阅读器、翻译切换、主题切换、HTML 安全过滤和图片请求边界测试。
